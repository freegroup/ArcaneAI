# ArcaneAI

A text-based adventure game powered by AI. Create interactive stories with a visual state editor and let players explore your world through natural language conversations.

## Documentation

📖 **[Full Documentation](https://freegroup.github.io/ArcaneAI/)**

## Features

- 🎮 AI-powered text adventure gameplay
- 🗺️ Visual state machine editor for story design
- 🎤 Voice input/output support
- 🎵 Dynamic sound effects
- 📦 Inventory system
